## **IRSSI**

```
sudo apt install irssi
```

```
irssi
```

```
/connect -ssl irc.libera.chat
```

Change username 

```
/nick Dcypher
```

```
/set user_name Dcypher
```

```
/set real_name Dcypher
```

```
/save
```

Now #channelname means that we can join a channel if it doesnt exist then it means we can create a channel with this name

```
/join #DickCipher
```

Only admin can kick out the user and promote others to admin

```
/op username -> admin
```

```
/kick username -> kick
```

```
/exit
```
